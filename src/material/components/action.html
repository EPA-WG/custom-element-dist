cem-action is button-looking component from <custom-element> Material suite based on the Material Design specification for buttons.
Action triggers the flow by changing the data slice or following the href attribute URL. Inspired by MUI

Attributes

text optional attribute defines the text inside the button.
class specific: cem-bend-sharp, cem-bend-smooth(default), cem-bend-round.

slice TBD

Corner shape is defined by --cem-bend
